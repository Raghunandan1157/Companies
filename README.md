## Visualisations

- **KPI Cards** — Sale, Bills, Qty, ATV, UPT, ASP at a glance
- **Target vs Achievement Gauge** — Semi-circle gauge showing 11% achievement
- **Sales Comparison Bar** — Horizontal bar comparing today's sale, MTD, trend, and target
- **MTD Stacked Bar** — Stacked bars showing achieved vs remaining for MTD and target
- **Per-Transaction Doughnut** — ATV, ASP, and UPT proportional breakdown

## How to Open

Open `index.html` in any modern browser. No build step or server required.

Uses [Chart.js](https://www.chartjs.org/) loaded from CDN.
